Quick Start

  1. Allow Location access on first launch.
  2. Tap Add Bearing to record a compass bearing at your current location.
  3. Repeat from new locations; color-code bearings to visualize convergence.
  4. Switch map sources: USGS / OpenStreetMap / Apple Maps.
  5. Cache maps (Menu → Cache Map) before going offline.

Troubleshooting

No location / blue dot missing
Check iOS Settings → Privacy & Security → Location Services → Fox Hunt Toolkit → set to While Using the App.

Compass seems inaccurate
Calibrate by moving the device in a figure-8. Avoid magnets/metal nearby.

Offline tiles not showing
Ensure you cached the area before going offline (Menu → Cache Map). Some providers require an API key or plan that allows offline use.

Data & Privacy

We don’t require an account. Bearings and cached tiles stay on your device. See our Privacy Policy.
